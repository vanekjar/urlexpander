PUBLIC API

URL EXPANDER offers an easy to use REST API to unshorten URLs from different shortening services. This API was tested with a lot of different services like goo.gl, bitly.com, tinyurl.com, but it should work as well with the others.

API request

GET http://urlexpander.tk/api/expand?url=[short-url]

Replace [short-url] with shortened url.

Success response

200 OK
{"original":"", "expanded":""}

Result is a JSON message containing the following fields:
original: Original (shortened) URL
expanded: Expanded URL

Error response

400 Bad Request || 500 Internal Server Error
{"error":""}

Result is a JSON message containing error details.

About Url Expander

Url Expander is a free to use service for expanding shortened urls from various services. Source code is available on GitHub under free licence.
